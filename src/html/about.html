The Maze Game

A simple yet fun game to play when you're bored.

About The Maze Game

You can use the arrow keys or the four control buttons below the maze to help the player navigate through the maze.

The player's goal is to reach the end of the maze.

If the player reaches the end of the maze, the maze will regenerate and the player will be placed at the start of the new maze.

If you find yourself stuck, you can regenerate the maze by clicking the "Regenerate Maze" button.

Good luck!
